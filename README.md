# WebdriverIO
This Demo automation project uses TypeScript WebdriverIO with Mocha and Chai.

## installation
 clone repository and:
```sh
npm install
```

## Run Tests
compile:
```sh
tsc --build
```
and start test:
```sh
npx wdio run wdio.conf.ts
```

## Results
After test is finished results can be found under `./results` folder.
